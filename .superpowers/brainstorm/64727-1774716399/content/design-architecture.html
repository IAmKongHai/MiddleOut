Module Responsibilities

Module Framework Responsibility
AppDelegate AppKit App lifecycle, agent mode, relaunch detection, window management
HotkeyManager KeyboardShortcuts Register global hotkey, trigger processing pipeline
FinderBridge NSAppleScript Get selected file URLs from Finder
FileRouter Foundation Classify files by extension, dispatch to correct processor
ImageProcessor ImageIO / UTI Convert any supported image to compressed JPG
PDFProcessor PDFKit + ImageIO Render each PDF page to JPG, output to named folder
ProgressPanel AppKit (NSPanel) Floating progress bar, skip messages, auto-dismiss
SoundPlayer NSSound Play completion chime
SettingsView SwiftUI Hotkey binding, quality slider + presets, launch at login, about
SettingsStore @AppStorage Persist quality setting, hotkey state, login toggle
