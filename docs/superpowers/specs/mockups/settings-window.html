Design Notes

  • Window size: ~400x350pt, fixed size, centered on screen
  • Tabs: General (shortcuts + quality + login) and About (version + links)
  • Preset buttons + slider are linked: clicking a preset moves the slider; dragging the slider deselects presets if value doesn't match
  • Close window = hide (app stays alive in background)
  • Only way to quit is the red "Quit MiddleOut" button
